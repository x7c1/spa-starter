# SPA Starter Kit

Sample application to learn modern SPA development.

## Current Structure


```
.
├── config
│   ├── postcss.config.js
│   └── webpack.common.js
├── dist
│   ├── bundle.js
│   ├── bundle.js.map
│   ├── index.html
│   ├── media
│   │   └── owl.[hash].png
│   └── styles
│       ├── main.[hash].css
│       └── main.[hash].css.map
└── src
    ├── assets
    │   └── owl.png
    ├── index.css
    ├── index.html
    ├── index.js
    ├── index.scss
    └── palette.scss
```

## Building

To generate `./dist` directory, run following:

```
npm run build
```
